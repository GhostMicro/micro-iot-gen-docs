# ยินดีต้อนรับสู่ Micro-IoT Generator Docs

**Micro-IoT Generator** คือแพลตฟอร์มที่ช่วยให้การสร้างอุปกรณ์ IoT ด้วย ESP8266 และ ESP32 เป็นเรื่องง่ายที่สุด ด้วยระบบจัดการ Pin อัจฉริยะและการสร้างโค้ดอัตโนมัติ

## เริ่มต้นใช้งาน

ดูคู่มือ [การติดตั้ง](user-guide/installation.md) เพื่อเริ่มต้นโปรเจกต์แรกของคุณ

## ฟีเจอร์เด่น

*   **Intelligent Pin Allocator**: ไม่ต้องกลัวต่อสายผิด หรือ Pin ชนกัน
*   **25+ Modules**: รองรับเซนเซอร์ยอดนิยมครบถ้วน
*   **GRIDS-IOT-V1 Ready**: รองรับมาตรฐาน MQTT ทันทีที่เจนโค้ด
*   **OTA Support**: อัปเดตผ่าน WiFi ได้เลย
